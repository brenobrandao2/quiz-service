# Deployment
- Depois de commitar novas implementações na branch master, acesse o servidor e atualize a branch master na pasta do projeto (/var/www/quiz-adm.lifeandmoney.com.br/project/quiz-service-adm/) com o comando 'git pull'.
- Caso seja necessário, execute o restart do serviço no PM2 com o seguinte comando 'pm2 restart quiz-adm-service --watch'.
